# Toy Robot Simulator

The application is a simulation of a toy robot moving on a square tabletop, of dimensions 5 units x 5 units

## Setup

```bash
$ npm install
```

## Usage

Start development server:

```bash
$ npm start
```

Build production version:

```bash
$ npm run build
```

Run javascript (ESLint):

```bash
$ npm run lint
```

Run karma tests:

```bash
$ npm test
```
